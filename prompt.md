Use a free public news API. Something like this: 

```javascript
async function fetchArticles() {
  try {
    // Using TheNewsAPI.com - completely free, real articles, real images
    const response = await fetch(
      'https://api.thenewsapi.com/v1/news/top?language=en&limit=20'
    )
    
    const data = await response.json()
    
    return data.data.map(article => ({
      id: article.uuid,
      title: article.title,
      description: article.description || 'No description available',
      image_url: article.image_url || 'https://via.placeholder.com/400x300?text=No+Image',
      category: article.category || 'News',
      published_at: article.published_at
    }))
  } catch (error) {
    console.error('Error fetching articles:', error)
    return []
  }
}
```
That's it. Real articles from TheNewsAPI.com. Inform me if anything else is required.
